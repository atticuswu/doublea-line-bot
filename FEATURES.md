## 智能分類邏輯

DoubleA 會自動判斷每一則訊息的類型，不需要任何前綴或指令：

| 訊息類型 | 判斷條件 | 處理方式 |
|---------|---------|---------|
| 行事曆事件 | 有明確時間或日期 + 未來要發生的事 | 加入 Google Calendar |
| 待辦事項 | 有任務性質，但沒有具體時間 | 記錄到 Google Tasks |
| 修改行事曆 | 含有「修正/更改/調整/修改」且指向最近事件 | 更新 Google Calendar |
| 忽略 | 日常聊天、情感表達、問句、已發生的事 | 無動作 |

### 即時回饋
符合行事曆或待辦條件的訊息，Bot 會立即回覆「⏳ 收到！處理中，請稍候...」確認已在處理中。

### 範例

| 訊息 | 結果 |
|------|------|
| `明天下午3點帶女兒去看醫生` | 📅 加入行事曆 |
| `下週五晚上6點吃飯` | 📅 加入行事曆 |
| `明天早上7點半出發去台北，後天晚上10點回到玉里` | 📅 加入 2 個行事曆 |
| `記得幫女兒買新鞋` | 📌 加入待辦 |
| `芋頭卷要先訂2份` | 📌 加入待辦 |
| `我愛你老婆` | 忽略 |
| `今天好累` | 忽略 |

## 功能一：行事曆事件

### 自動觸發
在群組中說出含有時間的事情，DoubleA 會自動：

1. 建立 Google Calendar 事件（atticus.wu@gmail.com）
2. 邀請 Angel 加入行事曆（angelsu0923@gmail.com）
3. 將原始訊息全文存入行事曆附註欄
4. 回覆確認訊息（含標題、日期時間、地點、Calendar 連結）
5. 附上一句鼓勵語

### 多事件支援
一句話包含多個時間點，Bot 自動拆解並分別建立行事曆。

範例：「明天早上7點半出發去台北，後天晚上10點回到玉里」
→ 建立「出發去台北」（5/26 07:30）＋「回到玉里」（5/27 22:00）兩筆事件

### 自動提醒
活動開始前 **2 小時**，DoubleA 會主動推送提醒到群組。

### 修改行事曆
發送含有「修正/更改/調整/修改」的訊息，Bot 自動更新最近一筆事件的時間。

範例：「修正以上行事曆，把26號改成25號」、「時間改成下午4點」

### 範例回覆（單一事件）
```
📅 已加入行事曆！

【帶女兒去看醫生】
🗓 5月23日 15:00

🔗 https://www.google.com/calendar/event?eid=...

✅ Angel 已收到邀請
⏰ 將於開始前 2 小時提醒

🧠 記下來了！腦袋的空間留給更重要的事。
```

### 範例回覆（多個事件）
```
📅 已加入 2 個行事曆！

1.【出發去台北】
   🗓 5月26日 07:30
   📍 台北
   🔗 https://...

2.【回到玉里】
   🗓 5月27日 22:00
   📍 玉里
   🔗 https://...

✅ Angel 已收到邀請
⏰ 將於各活動開始前 2 小時提醒

📆 安排好了，就不用一直惦記著這件事了。
```

## 功能二：待辦事項

### 自動觸發
在群組中說出沒有時間的任務，DoubleA 會自動記錄到 Google Tasks 並回覆確認（含鼓勵語）。

### 指令

| 指令 | 說明 |
|------|------|
| `todo` 或 `待辦` | 列出所有未完成的待辦事項 |
| `del 1` | 將第 1 項標記為完成（並附鼓勵語） |
| `del 5` | 將第 5 項標記為完成（並附鼓勵語） |
| `完成 關鍵字` | 用關鍵字搜尋並標記完成（並附鼓勵語） |
| `done 關鍵字` | 同上（英文版） |

### 範例回覆（完成待辦）
```
✅ 已完成：【幫女兒買新鞋】

🌟 今天又完成一件事了，超棒的！
```

## 功能三：每日主動提醒

DoubleA 每天主動推送兩次日程摘要，不需要任何指令：

### 早上 8:00 — 早安摘要
- 今天所有行事曆事件（含時間）
- 所有未完成待辦清單

```
🌅 早安！今天的安排：

📅 今日行事曆

・09:00 帶女兒去看醫生
・18:00 Angel 回家

📋 待辦清單

1. 幫女兒買新鞋
2. 訂芋頭卷 2 份

🔗 https://tasks.google.com/
```

### 下午 5:00 — 傍晚提醒
- 今晚（17:00 以後）的行事曆事件
- 所有未完成待辦清單

若當天無任何行事曆及待辦事項，則靜音不發。

### 活動前提醒
活動開始前 **2 小時**，單獨推送提醒（每 15 分鐘檢查一次）。

## 技術架構

| 元件 | 技術 |
|------|------|
| 訊息接收 | LINE Messaging API（Webhook） |
| 智能分類 | Gemini 2.5 Flash API（JSON mode + response schema） |
| 行事曆 | Google Calendar API（讀寫） |
| 待辦事項 | Google Tasks API |
| 狀態儲存 | Google Firestore |
| 提醒排程 | Google Cloud Scheduler |
| 後端伺服器 | Python + FastAPI |
| 部署環境 | Google Cloud Run（24/7 常駐，min-instances=1） |
